# Global-Air-Qulaty
Global Air Quality Dataset
##Description

This Excel file contains global air quality data collected across different locations. The dataset is useful for environmental analysis, data visualization projects related to pollution and public health.

File: Global air qul.xlsx
Format: Microsoft Excel (.xlsx)

##Contents

The dataset includes the following information (typical columns may include):

Country / City / Location

Date / Time

AQI (Air Quality Index)

Air pollution analysis

Trend analysis over time
